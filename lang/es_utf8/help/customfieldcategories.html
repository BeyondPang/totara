Categorías de campo personalizadas

La opción Categorías de campo personalizadas le permite establecer categorías para mantener campos personalizados bajo un nivel de profundidad.

Las categorías de campo personalizadas y los campos personalizados se establecen para permitir que toda la información relevante de los elementos de la jerarquía esté reunida y aparezca en las páginas de "Agregar/Modificar elementos de jerarquía".

Los nombres de las categorías de campo personalizadas deben ser únicos en el nivel de profundidad. Tiene que tener al menos una categoría de campo personalizada para poder establecer campos personalizados.

Agregar una categoría personalizada: haga clic en Crear una categoría de campo personalizada para agregar una nueva categoría de campo personalizada.

Modificar/Eliminar una categoría personalizada: Haga clic en Activar modificación para modificar o eliminar una categoría de campo personalizada existente.
