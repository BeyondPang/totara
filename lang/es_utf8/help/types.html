Tipos

Types proporciona la forma de categorizar elementos individuales dentro de sus jerarquías. Cuando crea tipos, puede asignarle a ellos campos especializados, lo que le permitirá asignar metadatos a los elementos que pertenecen a ese tipo. Se pueden configurar tipos múltiples dentro de una jerarquía, cada uno con su propio conjunto de campos especializados. Esto le permitirá crear diferentes tipos de elementos.

Cuando cree un nuevo elemento en la jerarquía, puede asignarlo a cualquiera de los tipos disponibles o dejarlo sin clasificar. Si un elemento es asignado a un tipo, los campos personalizados definidos para ese tipo estarán disponibles cuando edite el elemento.
