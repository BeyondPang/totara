Casilla para marcar el progreso

Algunas actividades se muestran con casillas para marcar el progreso. Estas son una forma de realizar un seguimiento del progreso durante el curso.

Cuando vea una casilla con una marca punteada, puede hacer clic en ella para marcar la casilla. (Si cambia de opinión, puede marcarla de nuevo para eliminar la marca).

Estos iconos se ofrecen para que pueda realizar un seguimiento de su propio progreso, y son completamente opcionales. Se lo desea, puede marcar las casillas cuando piense que ha terminado alguna parte.

Algunos cursos pueden incluir casillas para seleccionar en blanco: espacios vacíos que no tienen una marca punteada. Usted no puede marcar estas casillas, se marcarán automáticamente de acuerdo con las condiciones establecidas al crear el sitio web del curso. Por ejemplo, un recurso puede marcarse automáticamente después de descargar el archivo. (Algunas veces tendrá que volver a cargar la página del curso para ver este tipo de cambio).

Algunos miembros del personal, como los profesores, pueden ver qué casillas ha marcado. Sin embargo, la información no se utiliza en la evaluación y no cuenta para su calificación del curso. No necesita marcar casillas si no quiere.
