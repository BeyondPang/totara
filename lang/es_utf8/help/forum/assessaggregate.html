Tipos de acumulación de foro

Acumulación de foro define cómo se combinan todas las calificaciones dadas a las publicaciones en un foro para formar la calificación final (para cada publicación y para toda la actividad del foro).

Elija entre los siguientes métodos de acumulación:

  • Media (predeterminado)

    La media de todas las calificaciones dadas a las publicaciones en ese foro. Es particularmente útil con la calificación de compañeros cuando se están realizando múltiples calificaciones.

  • Recuento

    El número de publicaciones calificadas se convierte en la calificación final. Es útil cuando el número de publicaciones es importante. Tenga en cuenta que el total no puede exceder la calificación máxima permitida para el foro.

  • Máxima

    La calificación más alta es la calificación final. Este método es útil para enfatizar el mejor trabajo de los participantes, permitiéndoles publicar publicaciones de alta calidad así como un número de respuestas más informales a otros.

  • Mínima

    La calificación mínima es la calificación final. Este método promueve una cultura de alta calidad para todas las publicaciones.

  • Suma

    Se suman todas las calificaciones de un usuario particular. Tenga en cuenta que el total no puede exceder la calificación máxima del foro.
