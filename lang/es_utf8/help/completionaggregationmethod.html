Método de acumulación

Si el Método de acumulación se establece en la opción Todos, el curso se marcará como realizado cuando el usuario cumpla todos los criterios establecidos para la realización del curso.

Si el Método de acumulación se establece en la opción Algunos, el curso se marcará como realizado cuando el usuario cumpla alguno de los criterios establecidos para la realización del curso.
