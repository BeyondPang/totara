Realización manual por: Método de acumulación

Cuando está establecido el Método de acumulación como Todas cada una de las funciones seleccionadas debe marcar que el estudiante ha realizado el curso para cumplir los criterios.

Cuando está establecido el Método de acumulación como Algunas solo una de las funciones seleccionadas debe marcar que el estudiante ha realizado el curso para cumplir los criterios.
