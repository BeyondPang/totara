Actividades realizadas: método de acumulación

Cuando se establece el Método de acumulación en la opción Todas, el usuario tiene que completar todas las actividades seleccionadas para marcarlas como realizadas para estos criterios.

Cuando se establece el Método de acumulación en la opción Algunas, el usuario tiene que completar alguna de las actividades seleccionadas para marcarlas como realizadas para estos criterios.
