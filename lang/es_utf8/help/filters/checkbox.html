Filtro de casilla de verificación

Este filtro le permite filtrar la información basándose en una serie de casillas de verificación. El filtro tiene las opciones siguientes:

  • es cualquier valor: esta opción desactiva el filtro (es decir, este filtro acepta toda la información)
  • coincide con alguna selección: esta opción permite la información si coincide con cualquiera de las opciones marcadas
  • coincide con todas las selecciones: esta opción permite la información si coincide con todas las opciones marcadas
