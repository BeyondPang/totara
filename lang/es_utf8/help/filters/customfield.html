Filtro de campo personalizado

Este filtro le permite filtrar las competencias basándose en los valores de los campos personalizados. El filtro se puede aplicar a un único campo personalizado o a todos los campos personalizados. El filtro tiene las opciones siguientes:

  • contiene: esta opción solo permite las competencias para las que el campo especificado contenga el texto introducido (si no se ha introducido texto, el filtro se desactiva)
  • no contiene: esta opción solo permite las competencias para las que el campo especificado no contenga el texto introducido (si no se ha introducido texto, el filtro se desactiva)
  • es igual a: esta opción solo permite las competencias para las que el campo especificado sea igual al texto introducido (si no se ha introducido texto, el filtro se desactiva)
  • comienza con: esta opción solo permite las competencias para las que el campo especificado comience con el texto introducido (si no se ha introducido texto, el filtro se desactiva)
  • termina con: esta opción solo permite las competencias para las que el campo especificado termine con el texto introducido (si no se ha introducido texto, el filtro se desactiva)
  • está vacío: esta opción solo permite las competencias para las que el campo especificado sea igual a una cadena vacía, pero está definido (el texto introducido se ignora)
  • no está definido: esta opción solo permite las competencias para las que el campo especificado no esté definido (el texto introducido se ignora)
  • está definido: esta opción solo permite las competencias para las que el campo especificado esté definido (el texto introducido se ignora)
