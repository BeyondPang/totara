Filtro número

Este filtro le permite filtrar la información numérica basándose en su valor. El filtro tiene las opciones siguientes:

  • es igual a: esta opción solo permite la información que sea igual al texto introducido (si no se ha introducido texto, el filtro se desactiva)
  • no es igual a: esta opción solo permite la información que no sea igual al texto introducido (si no se ha introducido texto, el filtro se desactiva)
  • es mayor que: esta opción solo permite la información que tenga un valor numérico mayor que el texto introducido (si no se ha introducido texto, el filtro se desactiva)
  • es mayor que: esta opción solo permite la información que tenga un valor numérico mayor que el texto introducido (si no se ha introducido texto, el filtro se desactiva)
  • es menor que: esta opción solo permite la información que tenga un valor numérico menor que el texto introducido (si no se ha introducido texto, el filtro se desactiva)
  • es mayor o igual que: esta opción solo permite la información que tenga un valor numérico mayor o igual que el texto introducido (si no se ha introducido texto, el filtro se desactiva)
  • es menor o igual que: esta opción solo permite la información que tenga un valor numérico menor o igual que el texto introducido (si no se ha introducido texto, el filtro se desactiva)
