Cargar fotografías de usuario

Las fotografías de usuario se pueden cargar como archivo zip de archivos de imágenes. Los archivos de imágenes deben llamarse atributo-usuario-elegido.extensión. Por ejemplo, si el atributo de usuario elegido para que coincida con las imágenes es el nombre del usuario y este es usuario1234, el nombre de archivo debería ser usuario1234.jpg.

Los tipos de fotografías compatibles incluyen gif, jpg y png.

Los nombres de los archivos de fotografía no distinguen entre mayúsculas y minúsculas.
