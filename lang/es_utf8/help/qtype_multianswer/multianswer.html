Respuestas incluidas (Cloze)

Este tipo de preguntas muy flexibles es parecido al famoso formato conocido como Cloze.

Las preguntas constan de pasajes de texto (en formato Moodle) que tiene varias subpreguntas incluidas, entre las que se encuentran:

  • respuestas cortas (RESPUESTA CORTA o RC o MW), las mayúsculas o minúsculas no son importantes,
  • respuestas cortas (RESPUESTA CORTA_C o RCC o MWC), las mayúsculas o minúsculas deben coincidir,
  • respuestas numéricas (NUMÉRICA o NM),
  • múltiples opciones (MÚLTIPLES OPCIONES o MO), representadas como menú desplegable en línea en el texto
  • múltiples opciones (MÚLTIPLES OPCIONES_V o MOV), representadas como columna vertical de botones radiales, o
  • múltiples opciones (MÚLTIPLES OPCIONES_H o MOH), representadas como fila horizontal de botones radiales.

Actualmente no hay interfaz gráfica para crear estas preguntas; necesita especificar el formato de la pregunta utilizando el cuadro de texto o importándolo de archivos externos.
