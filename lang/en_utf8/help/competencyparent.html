Parent Competency

Parent competency allows you to create parent/child relationships between competencies in the competency framework using depth levels.

Select the Parent competency from the dropdown menu. Select Top if you want the competency to sit at the top level of the hierarchy.

When there is only one depth level set up in the competency framework the parent competency is set at Top.

Note: to set up parent/child relationships you need to have at least two depth levels set up.
