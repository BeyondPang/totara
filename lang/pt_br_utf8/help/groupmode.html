Modo de grupo

O modo de grupo pode ser de três níveis:

  • Nenhum grupo - não existem subgrupos, todo mundo faz parte de uma grande comunidade
  • Grupos separados - cada grupo só pode ver o seu próprio grupo, os outros são invisíveis
  • Grupos visíveis - cada grupo trabalha no seu próprio grupo, mas também pode ver os outros

O modo de grupo pode ser definido em dois níveis:

1. Nível do curso

O modo de grupo definido no nível do curso é o modo padrão para todas as atividades definidas dentro desse curso

2. Nível da atividade

Cada atividade que suporta os grupos também pode definir seu próprio modo de agrupamento. Se o curso for definido como "forçar modo do grupo", a configuração para cada atividade é ignorada.
