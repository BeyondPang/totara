Atividades concluídas: Método de agregação

Quando o Método de agregação está configurado como Todos, o aluno precisa concluir todas as atividades selecionadas para ser marcado como concluído para estes critérios.

Quando o Método de agregação está configurado como Qualquer, o aluno precisa concluir qualquer uma das atividades selecionadas para ser marcado como concluído para estes critérios.
