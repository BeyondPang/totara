Tipos

Os tipos proporcionam uma forma de categorizar os itens individuais dentro das suas hierarquias. Quando você criar os tipos, poderá determinar campos personalizados a estes, que permitirão determinar metadados a itens pertencentes a este tipo. Múltiplos tipos poderão ser configurados dentro da hierarquia, cada um com seus próprios campos personalizados. Isto permitirá que crie diferentes tipos de itens.

Quando você criar um novo item de hierarquia, você poderá determiná-lo para qualquer dos tipos disponíveis ou deixá-los sem clasificação. Se um item for para um determinado tipo, os campos personalizados definidos para esse tipo, estarão disponíveis quando realizar a edição do item.
