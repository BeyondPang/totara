Ocultar nomes dos alunos

Uma atribuição classificada pelos colegas pode ser classificada de maneira anônima. Nesse caso, os nomes (e fotos) dos alunos que fazem a classificação não são mostrados. Apenas os nomes de arquivo dos envios são usados para identificar o trabalho classificado.

Quando a atribuição classificada pelo colega não é anônima, os trabalhos são mostrados com os nomes (e fotos) dos alunos que enviaram o trabalho. Isso pode levar a um viés nas classificações.

Observe que se as notas do professor forem mostradas para os alunos, elas nunca são anônimas.
