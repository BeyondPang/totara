Filtro do número

Esse filtro permite filtrar as informações numéricas com base no seu valor. O filtro tem as seguintes opções:

  • é igual a - essa opção permite apenas as informações para as quais o campo especificado é igual ao texto inserido (se nenhum texto for inserido, o filtro é desativado)
  • não é igual a - essa opção permite apenas as informações para as quais o campo especificado não são iguais ao texto inserido (se nenhum texto for inserido, o filtro é desativado)
  • é maior que - essa opção permite apenas as informações que tenham valor numérico maior que o texto inserido (se nenhum texto for inserido, o filtro é desativado)
  • é maior que - essa opção permite apenas as informações que tenham valor numérico maior que o texto inserido (se nenhum texto for inserido, o filtro é desativado)
  • é menor que - essa opção permite apenas as informações que tenham valor numérico menor que o texto inserido (se nenhum texto for inserido, o filtro é desativado)
  • é maior ou igual a - essa opção permite apenas as informações que tenham valor numérico maior ou igual ao texto inserido (se nenhum texto for inserido, o filtro é desativado)
  • é menor ou igual a - essa opção permite apenas as informações que tenham valor numérico menor ou igual ao texto inserido (se nenhum texto for inserido, o filtro é desativado)
