Filtro do campo personalizado

Esse filtro permite filtrar as competências com base nos valores dos campos personalizados. O filtro pode ser aplicado em um único campo personalizado ou em todos. O filtro tem as seguintes opções:

  • contém - essa opção permite apenas as competências para as quais o campo especificado contém o texto inserido (se nenhum texto foi inserido, o filtro é desativado)
  • não contém - essa opção permite apenas as competências para as quais o campo especificado não contém o texto inserido (se nenhum texto foi inserido, o filtro é desativado)
  • é igual a - essa opção permite apenas as competências para as quais o campo especificado é igual ao texto inserido (se nenhum texto for inserido, o filtro é desativado)
  • começa com - essa opção permite apenas as competências para as quais o campo especificado começa com o texto inserido (se nenhum texto for inserido, o filtro é desativado)
  • termina com - essa opção permite apenas as competências para as quais o campo especificado termina com o texto inserido (se nenhum texto for inserido, o filtro é desativado)
  • é vazio - essa opção permite apenas as competências para as quais o campo especificado é igual a uma cadeia de caracteres vazia, mas está definido (o texto é ignorado)
  • não é definido - essa opção permite apenas as competências para as quais o campo especificado não é definido (o texto é ignorado)
  • é definido - essa opção permite apenas as competências para as quais o campo especificado é definido (o texto inserido é ignorado)
