Conclusão manual até: Método de agregação

Quando o Método de agregação está configurado como Todos, cada uma das funções selecionadas deve marcar o aluno como concluído para cumprir os critérios

Quando o Método de agregação está configurado como Qualquer, apenas as funções selecionadas devem marcar o aluno como concluído para cumprir os critérios.
