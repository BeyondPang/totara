Segurança do navegador

Essa opção oferece várias maneiras de tentar restringir como os alunos podem tentar 'trapacear' enquanto tentam resolver um questionário. No entanto, isso não é um problema simples, e o que em uma situação é considerado 'trapaça' pode, em outra situação, ser um uso efetivo da tecnologia da informação. (Por exemplo, a capacidade de encontrar rapidamente as respostas usando um mecanismo de busca).

Observe também que isso não é somente um problema de tecnologia com uma solução técnica. A trapaça é mais antiga que os computadores, e embora os computadores facilitem certas ações como copiar e colar, eles também facilitam para o professor a detecção da trapaça, por exemplo usando os relatórios do questionário. As opções fornecidas aqui não são infalíveis, e embora elas dificultem algumas formas de trapaça para os alunos, elas também tornam mais inconveniente para eles tentarem responder os questionários.

Você também deve considerar outras maneiras de dificultar a trapaça nos seus questionários:

  • Você pode usar um grande banco de perguntas, e o questionário faz uma seleção aleatória das perguntas, para que diferentes alunos vejam perguntas diferentes, porém semelhantes.
  • Você pode usar a opção de variar as respostas, para que a resposta certa da pergunta 1 não seja sempre a opção A.
  • Você pode fazer perguntas que exijam que o aluno analise as informações fornecidas e não apenas se lembre dos fatos.

    • Lembrando dos avisos acima, aqui está uma descrição das opções disponíveis.

    Nenhum

    Nenhum impedimento é colocado para os alunos que tentam responder o questionário.

    Pop-up de tela cheia com segurança JavaScript

    Existe um limite do que o questionário (executado em um servidor da Web) pode fazer para restringir o que o aluno pode fazer em seu computador enquanto o responde. No entanto, essa opção faz o que é possível:

    • O questionário só inicia se o aluno tiver um navegador da Web habilitado para JavaScript.
    • O questionário aparece em uma janela com pop-up de tela cheia que cobre todas as outras janelas e não possui controles de navegação.
    • Até onde possível, os alunos são impedidos de usar recursos como copiar e colar.

    Exigir o uso de um Navegador de exame seguro

    Essa opção só aparecerá se o administrador a ativou.

    Navegador de exame seguro é um navegador da Web personalizado que deve ser baixado um instalado no computador que o aluno usa para resolver o questionário. As restrições impostas são semelhantes às da janela pop-up, mas como o Navegador de exame seguro é um software executado no computador do aluno, ele é muito mais eficiente para restringir suas ações. Se você selecionar essa opção:

    • Os alunos só poderão responder o questionário se estiverem usando o Navegador de exame seguro.
    • A janela do navegador será em tela cheia (sem elementos de navegação).
    • A janela não pode ser fechada até que o teste seja enviado.
    • Teclas de atalhos como Win, Ctrl+Alt+Del, Alt+F4, F1, Ctrl+P, Printscreen, são desativadas.
    • Copiar, colar e o menu de contexto são desativados.
    • Não é possível mudar para outros aplicativos.
    • É proibido acessar outros sites.
