Gerenciar competências

Competências são as capacidades, conhecimentos e comportamentos específicos que você espera que a equipe adquira.

Selecione uma estrutura diferente: Use o menu suspenso Trocar estrutura para trocar as estruturas de competência.

Adicionar uma nova competência: Clique em Adicionar nova competência para adicionar uma nova competência à estrutura de competência selecionada.

Editar ou excluir uma competência existente: Clique em Ativar edição para editar ou excluir uma competência existente.
