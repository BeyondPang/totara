Pakettitiedostot

Paketti on tiedosto, jolla on zip- (tai pif)- pääte ja joka sisältää valideja AICC- tai SCORM-kurssitiedostoja.

SCORM-paketissa tulee olla zipin juuressa tiedosto nimeltä imsmanifest.xml, joka määrittelee SCORM-kurssin rakenteen, materiaalien sijainnit ja muita seikkoja.

AICC-paketin määrittelee useampi tiedosto (4-7), joila on tietty tiedostopääte. with defined extensions. Alla päätteiden kuvaukset:

  • CRS - kurssin kuvaus (Course Description file) (pakollinen)
  • AU - Assignable Unit file (pakollinen)
  • DES - kuvaus (Descriptor file) (pakollinen)
  • CST - kurssin rakenne (Course Structure file) (pakollinen)
  • ORE - Objective Relationship file (vapaaehtoinen)
  • PRE - edellytykset (Prerequisites file) (vapaaehtoinen)
  • CMP - suoritusvaatimukset (Completition Requirements file) (vapaaehtoinen)
