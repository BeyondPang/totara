Metodo di spacchettamento

Se il metodo di spacchettamento è impostato su Manifesta, allora l'oggetto SCORM/AICC associato deve essere un file ZIP con il file Manifest all'interno. Se un metodo di spacchettamento è impostato a un URL diretto, allora questo URL è l'URL di avvio per una singola attività AICC. Un pacchetto psuedo viene costruito intorno a questo e la modalità di comunicazione viene definita su AICC HACP.
