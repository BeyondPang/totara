Presentazione delle descrizioni dell'esercizio

Il docente deve inviare almeno una descrizione dell'esercizio o del compito. La descrizione può essere scritta in un documento Word o file HTML (o altro tipo di file visualizzabile in un browser). Questo file viene visualizzato dagli studenti e dovrebbe contenere le istruzioni per completare l'esercizio.

Il docente potrebbe presentare più descrizioni per un esercizio. Queste saranno usate casualmente e gli studenti visualizzeranno le varie versioni dell'esercizio. Tali varianti devono essere simili al Modulo di valutazione usato per valutare il lavoro di queste istruzioni.
