Filtro campo personalizzato

Questo filtro permette di filtrare le competenze in base ai valori dei campi personalizzati. Il filtro può essere applicato su un campo personalizzato singolo o su tutti i campi personalizzati. Il filtro ha le seguenti opzioni:

  • contiene - questa opzione permette soltanto le competenze per cui il campo specificato contiene il testo inserito (se non è stato inserito testo, il filtro è disabilitato)
  • non contiene - questa opzione permette soltanto le competenze per cui il campo specificato non contiene il testo inserito (se non è stato inserito testo, allora il filtro è disabilitato)
  • è uguale a - questa opzione permette soltanto le competenze per cui il campo specificato è uguale al testo inserito (se non è inserito testo, il filtro è disabilitato)
  • inizia con - questa opzione permtte soltanto le competenze per cui il campo specificato inizia con il testo inserito (se non è inserito alcun testo, il filtro è disabilitato)
  • finisce con - questa opzioen consente solo le competenze per cui il campo specificato finisce con il testo inserito (se non è inserito testo, il filtro è disabilitato)
  • è vuoto - questa opzione permette soltanto le competenze per cui il campo specificato è uguale a una stringa vuota, ma è definito (il testo inserito è ignorato)
  • non è definito - questa opzione permette soltanto le competenze per cui il campo specificato non è definito (il testo inserito è ignorato)
  • è definito - questa opzione permette soltanto le competenze per cui il campo specificato + definito (il testo inserito è ignorato)
