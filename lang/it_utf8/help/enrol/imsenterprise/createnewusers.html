Creazione automatica degli account utente

I dati di registrazione IMS Enterprise descrivono tipicamente un set di utenti. Se questa impostazione è attivata, gli account possono essere creati per qualsiasi utente che non si trova nel database di Moodle.

La ricerca di utenti è fatta per "idnumber" e per nome utente di Moodle.

Password

Le password non sono importate dal plugin di IMS Enterprise. Si raccomanda di usare i plugin di autenticazione di Moodle per autenticare gli utenti.
