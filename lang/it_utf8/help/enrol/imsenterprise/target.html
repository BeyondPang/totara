Target

Un file di dati IMS Enterprise può essere inteso per più "target" (ovverso diversi LMS, o diversi sistemi all'interno di una scuola o università. E' possibile specificare nel file Enterprise che i dati sono intesti per uno o più sistemi denominati sistemi di target, denominandoli nei tag <target> contenuti nel <proprietà>.

In molti casi, non è necessario preoccuparsene. Se si lascia l'impostazione di configurazione vuota, Moodle elaborerà il file di dati, che sia stato specificato un target oppure no. In alternativa, inserire il nome che verrà emesso nel tag <target>.
