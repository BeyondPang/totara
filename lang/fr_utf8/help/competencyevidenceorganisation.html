Organisation lors de l'achèvement

Cette option enregistre l'organisation dans laquelle l'utilisateur figurait lors de l'achèvement de l'élément de preuve de compétence. Dans la plupart des cas, il s'agira de l'organisation actuelle de l'utilisateur. Les utilisateurs changeant d'organisation au cours du temps, cette option permet de garder une trace du lieu où les utilisateurs se trouvaient lors de l'achèvement. Ce champ est optionnel.
