Organisation parent

Une organisation parent vous permet de gérer les relations parent/enfant entre les organisations.

Choisissez Organisation parent du menu déroulant. Choisissez Top si vous voulez que l'organisation soit au sommet de la hiérarchie.

Si vous changez l'organisation parent d'un objet, celui-ci sera déplacé pour se situer sous son nouveau parent, ainsi que tous ses enfants.

Note : pour configurer les relations parent/enfant vous devez avoir au moins un objet dans le cadre. Sinon l'option ne figurera pas.
