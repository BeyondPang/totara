Bloc d'entête du rapport

Le bloc d'entête du rapport est le bloc de données à propos d'un utilisateur spécifique, qui peut être inclus dans plusieurs pages du site (par example 'enregistrements d'apprentissage' ou 'planificateur de développement'). Il contient des informations sur l'utilisateur telles que son nom et des détails personnls.

Les contenus exacts du bloc peuvent être définis en ajoutant, supprimant et réarrangeant des champs dans ce formulaire.

Ajout d'objets :
Pour ajouter un objet, sélectionnez le champ que vous voulez à partir du menu déroulant appelé 'Choisissez une autre colonne...'. Entrez un entête de colonne, qui apparaîtra en tant que titre à côté de la valeur du champ. Vous pouvez aussi spécifier une valeur qui apparaîtra si aucune donnée pour ce champ n'est trouvée. Cliquez sur 'Enregistrer les modifications' puis répétez autant de fois que vous souhaitez ajouter d'objets.

Modifier objets :
Pour modifier l'entête ou le texte de données manquantes pour un objet, modifiez les champs texte puis cliquez sur 'enregistrer les modifications'. Vous serez renvoyé au formulaire avec les nouvelles valeurs.

Supprimer objets :
Cliquez sur la croix rouge à côté du nom de l'objet pour le supprimer du bloc d'entête. Il vous sera demandé de confirmer ce choix.

Réarranger objets :
Vous pouvez modifier l'ordre dans lequels les objets apparaissent dans la liste en cliquant sur les flèches haut et bas à côté de chaque objet.
