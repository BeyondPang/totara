Bloc d'entête du rapport

Le bloc d'entête du rapport est un bloc de données sur un utilisateur spécifique, qui peut être inclus sur de nombreuses pages à travers le site (par exemple les pages 'Historique d'apprentissage' ou 'Planificateur de développement'). Il contient certaines informations sur l'utilisateur telles que leur nom et leurs données personnelles

Le contenu exact du bloc est défini en ajouter, supprimant et réarrangeant les champs de ce formulaire.

Ajout d'objets :
Pour ajouter un objet, choisissez le champ que vous voulez dans le menu déroulant avec le lable 'Sélectionnez une autre colonne...'. Entrez un entête de colonne, qui apparaîtra comme titre à côté de la valeur du champ. Vous pouvez aussi spécifier une valeur qui apparaîtra si aucune donnée n'est trouvée pour cet utilisateur. Cliquez sur 'Enregistrer les modifications' puis répétez pour créer plusieurs objets.

Modifier des objets :
Pour modifier l'entête ou le texte par défaut pour un objet, modifiez les champs texte puis cliquez sur le bouton 'Enregistrer les modifications'.

Supprimer des objets :
Cliquez sur la croix rouge à côté des noms des objets pour les supprimer du bloc d'entête. Une confirmation de votre choix vous sera demandée.

Réarranger les objets :
Vous pouvez modifier l'order d'apparition des objets dans la liste en cliquant sur les flèches haut ou bas à côté de chaque objet.
