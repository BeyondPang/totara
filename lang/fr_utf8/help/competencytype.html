Type de compétence

Les administrateurs peuvent créer et attribuer des types de compétences. Si un type est attribué à une compétence, celle-ci hérite de tous les champs personnalisés qui ont été attribués à ce type. Cela vous permet d'organiser les métadonnées liées à vos compétences et de montrer seulement les champs permettant de trier les compétences.
