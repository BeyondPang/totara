Filtre de case à cocher

Ce filtre permet de filtrer les informations selon un ensemble de cases à cocher. Le filtre a les options suivantes:

  • de toute valeur - cette option désactive le filtre (le filtre accepte toutes les options)
  • correspond à une partie de la sélection - cette option autorise les informations correspondant à au moins une des options sélectionnées
  • correspond à toute la sélection - cette option autorise les informations correspondant à toutes les options sélectionnées
