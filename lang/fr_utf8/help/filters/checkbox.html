Filtre de case à cocher

Ce filtre vous permet de filtrer les informations selon une combinaison de cases à cocher. Le filtre possède les options suivantes :

  • pour toute valeur - cette option désactive le filtre (toutes les informations sont acceptées par ce filtre)
  • correspond à une partie de la sélection - cette option autorise les informations correspondant à au moins une des options cochées
  • correspond à toute la sélection - cette option autorises les informations correspondant à toutes les options cochées
