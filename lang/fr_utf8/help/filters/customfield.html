Filtre de champ personnalisé

Ce filtre vous permet de filtrer les compétences selon les valeurs de champs personnalisés. Ce filtre peut s'appliquer à un seul ou tous les champs personnalisés. Le filtre possède les options suivantes :

  • contient - cette option n'autorise que les compétences pour lesquelles le champ spécifié contient le texte entré (le filtre est désactivé si aucun texte n'est entré)
  • ne contient pas - cette option n'autorise que les compétences pour lesquelles le champ spécifié ne contient pas le texte entré (le filtre est désactivé si aucun texte n'est entré)
  • est égal à - cette option n'autorise que les compétences pour lesquelles le champ spécifié correspond exactement au texte entré (le filtre est désactivé si aucun texte n'est entré)
  • commence par - cette option n'autorise que les compétences pour lesquelles le champ spécifié commence par le texte entré (le filtre est désactivé si aucun texte n'est entré)
  • se finit par - cette option n'autorise que les compétences pour lesquelles le champ spécifié finit par texte entré (le filtre est désactivé si aucun texte n'est entré)
  • est vide - cette option n'autorise que les compétences pour lesquelles le champ spécifié est laissé vide, mais est défini (le texte entré est ignoré)
  • n'est pas défini - cette option n'autorise que les compétences pour lesquelles le champ spécifié n'est pas défini (le texte entré est ignoré)
  • est défini - cette option n'autorise que les compétences pour lesquelles le champ spécifié est défini (le texte entré est ignoré)
