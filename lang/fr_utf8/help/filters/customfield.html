Filtre de champ personnalisé

Ce filtre vous permet de filtrer les compétences selon les valeurs de champs personnalisés. Ce filtre peuut être appliqué sur un seul champ personnalisé ou sur tous. Ce filtre possède les options suivantes:

  • contient - cette option n'autorise que les compétences pour lesquelles le champ personnalisé contient le texte entré (si aucun texte n'est entré, ce champ est désactivé)
  • ne contient pas - cette option n'autorise que les compétences pour lesquelles le champ personnalisé ne contient pas le texte entré (si aucun texte n'est entré, ce champ est désactivé)
  • est égal à - cette option n'autorise que les compétences pour lesquelles le champ personnalisé est égal au texte entré (si aucun texte n'est entré, ce champ est désactivé)
  • commence par - cette option n'autorise que les compétences pour lesquelles le champ personnalisé commence par le texte entré (si aucun texte n'est entré, ce champ est désactivé)
  • finit par - cette option n'autorise que les compétences pour lesquelles le champ personnalisé finit par le texte entré (si aucun texte n'est entré, ce champ est désactivé)
  • est vide - cette option n'autorise que les compétences pour lesquelles le champ personnalisé est vide mais défini (le texte entré est ignoré)
  • est non défini - cette option n'autorise que les compétences pour lesquelles le champ personnalisé est non défini (le texte entré est ignoré)
  • is defined - this option allows only competencies for which the specified field is defined (the text entered is ignored)
