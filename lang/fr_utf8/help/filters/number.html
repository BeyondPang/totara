Filtre numérique

Ce filtre vous permet de filtrer des informations numériques selon leur valeur. Le filtre possède les options suivantes :

  • est égal à - cette option n'autorise que les informations égales au texte entré (le filtre est désactivé si aucun texte entré)
  • n'est pas égal à - cette option n'autorise que les informations ne sont pas égales au texte entré (le filtre est désactivé si aucun texte entré)
  • est supérieur à - cette option n'autorise que les informations dont la valeur numérique est supérieure au texte entré (le filtre est désactivé si aucun texte entré)
  • est supérieur à - cette option n'autorise que les informations dont la valeur numérique est supérieure au texte entré (le filtre est désactivé si aucun texte entré)
  • est inférieur à - cette option n'autorise que les informations dont la valeur numérique est inférieure au texte entré (le filtre est désactivé si aucun texte entré)
  • est supérieur ou égal à - cette option n'autorise que les informations dont la valeur numérique est supérieure ou égale au texte entré (le filtre est désactivé si aucun texte entré)
  • est inférieur ou égal à - cette option n'autorise que les informations dont la valeur numérique est inférieure ou égale au texte entré (le filtre est désactivé si aucun texte entré)
