Filtres numériques

Ce filtre permet de filtrer des informations numériques selon leur valeur. Le filtre propose les options suivantes :

  • est égal à - cette option n'autorise que les informations égales au texte entré (si aucun texte n'est entré, ce filtre est désactivé)
  • n'est pas égal à - cette option n'autorise que les informations non égales au texte entré (si aucun texte n'est entré, ce filtre est désactivé)
  • est supérieur à - cette option n'autorise que les informations dont la valeur numérique est supérieure au texte entré (si aucun texte n'est entré, ce filtre est désactivé)
  • est supérieur à - cette option n'autorise que les informations dont la valeur numérique est supérieure au texte entré (si aucun texte n'est entré, ce filtre est désactivé)
  • est inférieur à - cette option n'autorise que les informations dont la valeur numérique est inférieure au texte entré (si aucun texte n'est entré, ce filtre est désactivé)
  • est supérieur ou égal à - cette option n'autorise que les informations dont la valeur numérique est supérieure ou égale au texte entré (si aucun texte n'est entré, ce filtre est désactivé)
  • est inférieur ou égal à - cette option n'autorise que les informations dont la valeur numérique est inférieure ou égale au texte entré (si aucun texte n'est entré, ce filtre est désactivé)
