Poste parent

Le poste parent permet de gérer les relations parent/enfant entre les postes.

Choisissez le Poste parent dans le menu déroulant. Choisissez Top si vous voulez que le poste soit au sommet de la hiérarchie.

Si vous modifiez le poste parent d'un objet, celui-ci ainsi que ses enfants seront déplacés en dessous du nouveau parent.

Note : pour configurer les relations parent/enfant, vous devez avoir au moins un objet dans le cadre. Autrement cette option ne figurera pas.
