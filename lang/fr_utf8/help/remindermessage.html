Message

Il s'agit du message de l'e-mail de rappel.

Le message peut inclure certains paramètres

  • [firstname] - Prénom utilisateur
  • [lastname] - Nom utilisateur
  • [coursepageurl] - Un lien cliquable vers la page du cours
  • [coursename] - Nom du cours
  • [managername] - Nom du gestionnaire utilisateurs
  • [days counter up] - Nombre de jours avant achèvement
  • [days count down] - Nombre de jours avant fin du délai
