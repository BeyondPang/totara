Gérer les organisations

Cette fonction vous permet de paramétrer les parties composantes de votre organisation dans la structure hiérarchique appropriée. Assurez-vous d'abord de sélectionner le cadre organisationnel.

Sélectionner un autre cadre : Utilisez Basculer de cadre pour changer de cadre organisationnel.

Ajouter une nouvelle organisation : Cliquez sur Ajouter une nouvelle organisation pour ajouter une nouvelle organisation au cadre organisationnel sélectionné.

Modifier ou supprimer une organisation existante : Cliquez sur Activer l'édition pour modifier ou supprimer une organisation existante.
