Dépendances du cours : méthode d'aggrégation

Quand la méthode d'aggrégation est réglée sur Tous l'apprenti doit avoir achevé tous les cours pour remplir le critère.

Quand la méthode d'aggrégation est réglée sur N'importe lequel l'apprenti doit avoir achevé au moins un cours pour remplir le critère.
