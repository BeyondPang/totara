Gérer les compétences

Les compétences sont les capacités, les savoirs et les comportements spécifiques que vous attendez de votre équipe.

Choisir un cadre différent : Utilisez le menu déroulant Changer de cadre pour changer de cadre de compétences.

Ajouter une nouvelle compétence : Cliquez sur Ajouter une nouvelle compétence pour ajouter une compétence au cadre de compétences actuel.

Modifier ou supprimer une compétence existante : Cliquez sur Activer l'édition pour modifier ou supprimer une compétence existante.
