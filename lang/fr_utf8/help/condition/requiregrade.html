Exiger note

Quand Exiger note est activé, l'apprenti doit avoir reçu une certaine note pour que l'activité soit considérée achevée.

Vous pouvez utiliser le livre de notes du cours pour configurer la note de passage de cette activité. Quand aucune note de passage n'est confiurée, l'activité sera marquée achevée quand l'apprenti recevra une note.

Distinguer les notes de passage et d'échec

Il est possible de configurer une activité pour qu'elle permette à l'apprenti de voir 'achevé, réussi' ou 'achevé, échoué' au lieu du standard 'achevé'. Ces résultats affichent une icône différente et un texte alternatif.

Configurer une note de passage sur une activité :

  1. Cliquez sur Notes du bloc 'Admin du cours' pour ouvrir le livre de notes du cours
  2. Sélectionnez Vue simple sous 'Catégories et objets' du menu déroulant 'Choisissez une action'.
  3. Cliquez sur Modifier à côté de l'objet de note pour lequel vous souhaitez définir une note de passage
  4. Cliquez sur Afficher avancés
  5. Entrez la note de passage dans le champ 'Note de passage'.
  6. Cliquez sur Enregistrer les modifications.

Une fois la note de passage configurée, les apprentis obtiendront soit 'achevé, réussi' soit 'achevé, échoué', après avoir reçu une note pour l'activité.

Note : pour activer la fonction permettant à l'utilisateur de voir soit 'achevé, réussi' soit 'achevé, échoué' vous devez sélectionner l'option 'Visibilité immédiate' pour les notes, sans quoi l'apprenti ne verra que le standard 'achevé'.
