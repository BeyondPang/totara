Note requise

Quand note requise est activé l'apprenti doit avoir reçu une note lors de l'activité pour que celle-ci soit considérée achevé.

Vous pouvez utiliser le livre de notes du cours pour régler les notes de passage de l'activité. Quand aucune note de passage n'a été configurée l'activité sera marquée achevée quand l'apprenti reçoit une note.

Distinguer les notes de passsage et d'échec

Il est possible de configurer une activité pour permettre à l'apprenti de voir 'achevé, réussi' ou 'achevé, échoué' au lieu du standard 'achevé'. Ces résultats possèdent des icônes et textes alternatifs différents.

Configurer une note de passage pour l'activité :

  1. Cliquez sur Notes du bloc 'Admin cours' pour ouvrir le livre de notes du cours
  2. Choisissez Vue simple sous 'Catégories et objets' du menu déroulant 'Choisissez une action...'.
  3. Cliquez sur le bouton Modifier à côté de l'objet de note pour l'activité pour laquelle vous souhaitez définir une note de passage
  4. Cliquez sur Afficher avancé
  5. Configurez la note de passage dans le champ 'Note de passage'.
  6. Cliquez sur Enregistrer les modifications.

Une fois que la 'note de passage' est configurée, l'apprenti devra atteindre un 'achevé, réussi' ou un 'achevé, échoué', quand ils achèvent l'activité.

Note : pour permettre à l'apprenti de voir 'achevé, réussi' ou 'achevé, échoué' les notes pour cette activité doivent être réglées sur 'visible immédiatement'. Sinon l'apprenti ne verra que "achevé".
