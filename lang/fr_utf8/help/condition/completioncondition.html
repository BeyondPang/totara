Conditions d'achèvement d'activité

Les Conditions d'achèvement d'activité vous permettent de restreindre la disponibilité d'un cours selon le statut d'achèvement d'autres cours.

Sélectionnez l'activité de cours du menu déroulant et sélectionnez le statut d'achèvement :

  • Doit être marqué achevé : l'activité sélectionnée doit être achevée pour que celle-ci soit disponible.
  • Ne doit pas être marqué achevé : l'activité sélectionnée ne doit pas être achevée pour que celle-ci soit disponible.
  • Doit être achevé avec note de passage : l'activité sélectionnée doit être achevée avec note de passage pour que celle-ci soit disponible
  • Doit être achevé avec échec : l'activité sélectionnée doit être achevée avec statut d'échec pour que celle-ci soit disponible.

Cliquez sur Ajouter 2 champs de condition au formulaire pour ajouter de nouveaux champs de condition d'achèvement d'activité.
