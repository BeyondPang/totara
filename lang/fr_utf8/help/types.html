Types

Les Types fournissent un moyen de catégoriser les objets dans vos hiérarchies. Quand vous crées des types vous pouvez leur attribuer des champs personnalisés, vous permettant de leur atrribuer des métadonnées appartenant au type. Plusieurs types peuvent être paramétrés au sein d'une hiérarchie, chacun avec ses champs personnalisés propres. Ceci vous permet de créer différents types d'objets.

Quand vous créez un nouvel objet de hiérarchie vous pouvez lui attribuer tout type disponible, ou le laisser non-classé. Si un objet est affecté à un type, les champs personnalisés pour ce type seront disponibles lors de la modification de l'objet.
