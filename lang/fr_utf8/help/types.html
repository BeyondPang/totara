Types

Les types sont un moyen de catégoriser les objets individuels au sein de vos hiérarchies. Quand vous créez des types, vous pouvez leur attribuer des champs personnalisés, ce qui vous permet d'attribuer des métadonnées aux objets appartenant à ce type. Plusieurs typespeuvent être configurés au sein d'une hiérarchie, chacun avec leurs propres champs personnalisés. Cela vous permet de créer différents types d'objets.

Quand vous créez un nouvel objet de hiérarchie vous pouvez lui assigner tout type disponible, ou le laisser non classé. Si un objet se voit assigner un type, les champs personnalisés définis pour ce type seront alors disponible lors de la modification de l'objet.
