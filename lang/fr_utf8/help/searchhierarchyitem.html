Recherche d'objet de hiérarchie

Entrez un ou plusieurs termes de recherche dans le champ textuel et appuyez sur recherche. Tous les résultats correspondants s'afficheront.

Un objet sera affiché si tous les mots entrés se trouvent dans le nom de l'objet, sa description, ou son numéro ID. La recherche n'est pas sensible à la casse, chercher "Compétence" permettra donc de trouver "COMPETENCE" ou "compétence".

Vous pouvez utiliser des apostrophes (') ou double apostrophes (") pour grouper des ensembles de mots. Dans ce cas, les mots doivent apparaître dans cet ordre.

Si vous voyez un menu déroulant, vous pouvez l'utiliser pour limiter les résultats à un seul cadre, ou chercher dans tous les cadres.
