Recherche d'objets de hiérarchie

Entrez un ou plusieurs termes de recherche dans le champ texte et appuyez sur Recherche. Tous les résultats correspondant seront affichés.

Un objet sera affiché si tous les mots entrés sonttrouvés quelque part dans le nom de l'objet, sa description ou son numéro ID. La recherche n'est pas sensible à la casse, et peut trouver "Compétence" aussi bien que "COMPETENCE" ou "compétence".

Vous pouvez utilisez les apostrophes simples (') ou doubles (") pour grouper un ensemble de mots. Dans ce cas, tous les mots doivent apparaître à la suite.

Si vous voyez un menu déroulant, vous pouvez l'utiliser pour limiter les résultats à un cadre, ou chercher dans tous les cadres.
