Méthode de décompactage

Si la méthode de décompactage est réglée sur Manifest, alors les objets SCORM/AICC associés doivent être un fichier ZIP avec les fichiers Manifest appropriés à l'intérieur. Si la méthode de décompactage est réglée sur URL directe, alors l'URL est considérée comme l'URL de démarrage d'une activité AICC. Un pseudo paquetage sera construit autour et le mode de communication sera considéré être AICC HACP.
