Gérer les organisations

Cette fonction vous permet de configurer les parties composantes de votre organisation dans la hiérarchie structurelle correcte.

Choisir un cadre différent : Utilisez Changer de cadre pour changer de cadre d'organisations.

Ajouter une nouvelle organisation: Cliquez sur Ajouter une nouvelle organisation pour ajouter une organisation au cadre d'organisations sélectionné.

Modifier ou supprimer une organisation existante : Cliquez sur Activer l'édition pour modifier ou supprimer une organisation existante.
