Gérer les organisations

Cette fonction vous permet de configurer les parties composantes de votre organisation dans une structure hiérarchique correcte.

Sélectionner un autre cadre : Utiliser Basculer de cadre pour changer de cadre organisationnel.

Ajouter un nouveau cadre : Cliquez sur Ajouter une nouvelle organisation pour ajouter une nouvelle organisation au cadre organisationnel sélectionné.

Modifier ou supprimer une organisation existante : Cliquez sur Activer l'édition pour éditer ou supprimer une organisation existante.
