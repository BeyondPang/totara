Taguer les cours

Taguer les cours vous permet d'ajouter tout tag moodle 'officiel' à votre cours, de manière à classifier ou identifier vos cours. Les tags officiels peuvent être ajoutés au site via la page Gérer les tags par les utilisateurs ayant les permissions appropriées.

Une fois configurés, les tags de cours sont affichés en tant que colonne et utilisés pour filtrer les résultats dans les rapports de cours.

Le tag de cours peut être désactivé au niveau site sur la page administrateur 'Paramètres de cours par défaut'.
