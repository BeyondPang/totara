Tag des cours

Le tag des cours vous permet d'ajouter n'importe lequel des tags moodle 'officiels' à votre cours, de sorte à classer ou identifier les cours. Les tags officiels peuvent être ajoutés au site via la page Gérer les tags par les utilisateurs possédant les permissions adéquates.

Une fois configurés, les tags de cours peuvent être affichés en colonne et utilisés pour filtrer les résultats dans les rapports de cours.

Le tag de cours peut être désactivé au niveau site depuis la page administrateur 'Paramètres par défaut du cours'.
