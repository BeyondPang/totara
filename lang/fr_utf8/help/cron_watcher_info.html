Observateur Cron

L'observateur Cron est un script situé sur 

<TOTARA>/admin/cron_watcher.php
Pour l'activer vous devez l'ajouter à la liste de cron de Linux ou créer une nouvelle tâche planifiée sous Windows.

Nous vous conseillons de régler le script pour qu'il soit exécuté toutes les 5 minutes.
