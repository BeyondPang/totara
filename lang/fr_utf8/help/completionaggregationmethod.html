Méthode d'aggrégation

Si la méthode d'aggrégation est réglée sur Tous le cours sera considéré achevé quand l'apprenti aura rempli tous les critères d'achèvement du cours.

Si la méthode d'aggrégation est réglée sur N'importe laquelle le cours sera considéré achevé quand l'apprenti aura rempli au moins un des critères d'achèvement du cours.
