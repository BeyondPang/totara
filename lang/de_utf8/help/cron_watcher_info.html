Cron Watcher

Cron Watcher ist ein Skript in 

<TOTARA>/admin/cron_watcher.php
Zum Aktivieren müssen Sie das Skript zur Liste der Cron-Jobs in Linux hinzufügen oder einen neuen geplanten Task in Windows erstellen.

Wir empfehlen, das Skript so einzustellen, dass es alle 5 Minuten läuft.
