Filter für benutzerdefiniertes Feld

Mit diesem Filter können Sie Kompetenzen auf Grundlage der Werte der benutzerdefinierten Feldern filtern. Der Filter kann für ein einzelnes benutzerdefiniertes Feld oder alle benutzerdefinierten Felder eingesetzt werden. Der Filter hat die folgenden Optionen:

  • enthält - diese Option erlaubt nur Kompetenzen, für die das festgelegte Feld den eingegebenen Text enthält (wenn kein Text eingegeben wird, dann ist der Filter deaktiviert)
  • enthält nicht - diese Option erlaubt nur Kompetenzen, für die das festgelegte Feld nicht den eingegebenen Text enthält (wenn kein Text eingegeben wird, dann ist der Filter deaktiviert)
  • ist gleich - diese Option erlaubt nur Kompetenzen, für die das festgelegte Feld genau dem eingegebenen Text entspricht (wenn kein Text eingegeben wird, dann ist der Filter deaktiviert)
  • beginnt mit - diese Option erlaubt nur Kompetenzen, für die das festgelegte Feld mit dem eingegebenen Text beginnt (wenn kein Text eingegeben wird, dann ist der Filter deaktiviert)
  • endet mit - diese Option erlaubt nur Kompetenzen, für die das festgelegte Feld mit dem eingegebenen Text endet (wenn kein Text eingegeben wird, dann ist der Filter deaktiviert)
  • ist leer - diese Option erlaubt nur Kompetenzen, für die das festgelegte Feld einem leeren String entspricht, aber definiert ist (der eingegebene Text wird ignoriert)
  • ist nicht definiert - diese Option erlaubt nur Kompetenzen, für die das festgelegte Feld nicht definiert ist (der eingegebene Text wird ignoriert)
  • ist definiert - diese Option erlaubt nur Kompetenzen, für die das festgelegte Feld definiert ist (der eingegebene Text wird ignoriert)
