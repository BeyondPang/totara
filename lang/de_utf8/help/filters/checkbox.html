Markierungsfeldfilter

Mit diesem Filter können Sie Informationen auf Grundlage eine Reihe von Markierungsfeldern filtern. Der Filter hat die folgenden Optionen:

  • ist beliebiger Wert - diese Option deaktiviert den Filter (d.h. alle Informationen werden durch diesen Filter angenommen)
  • entspricht einer gewählten Option - diese Option erlaubt Informationen, wenn diese einer der markierten Optionen entsprechen
  • entspricht allen gewählten Optionen - diese Option erlaubt Informationen, wenn diese allen markierten Optionen entsprechen
