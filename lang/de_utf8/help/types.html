Typen

Typen bieten die Möglichkeit der Kategorisierung von einzelnen Elementen innerhalb Ihrer Hierarchien. Wenn Sie Typen erstellen, dann können Sie diesen benutzerdefinierte Felder zuweisen, mit denen Sie Meta-Daten zu den Elementen, die zu diesem Typ gehören, zuweisen können. Innerhalb einer Hierarchie können mehrere Typen eingerichtet werden, jeweils mit eigenen benutzerdefinierten Feldern. Somit können Sie verschiedene Arten von Elementen erstellen.

Wenn Sie ein neues Hierarchieelement erstellen, können Sie es einem der verfügbaren Typen zuweisen oder unklassifiziert lassen. Wenn ein Element einem Typen zugewiesen wurde, sind die für diesen Typ festgelegten benutzerdefinierten Felder dann beim Bearbeiten des Elements verfügbar.
