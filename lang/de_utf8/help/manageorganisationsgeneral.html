Organisationen verwalten

Mit dieser Funktion können Sie die Bestandteile Ihrer Organisation für eine korrekte strukturelle Hierarchie einstellen. Stellen Sie zuerst sicher, dass Sie den korrekten Organisationstyp wählen.

Einen anderen Typ wählen: Verwenden Sie Typ wechseln, um die Organisationstypen zu wechseln.

Eine neue Organisation hinzufügen: Klicken Sie auf Neue Organisation hinzufügen, um dem gewählten Organisationstyp eine neue Organisation hinzuzufügen.

Eine bestehende Organisation bearbeiten oder löschen: Klicken Sie auf Bearbeiten einschalten, um eine bestehende Organisation zu bearbeiten oder zu löschen.
