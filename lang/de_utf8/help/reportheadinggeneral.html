Berichtsüberschriftenblock

Der Berichtsüberschriftenblock ist ein Block mit Daten zu einem bestimmten Nutzer, der in verschiedene Seiten auf der Website aufgenommen werden kann (z.B. auf den Seiten 'Lernverlauf' oder 'Entwicklungsplaner'). Er enthält Informationen über den Nutzer, z.B. Name und persönliche Daten.

Der genaue Inhalt des Blocks wird durch Hinzufügen, Entfernen und Neuanordnen der Felder in diesem Formular festgelegt.

Hinzufügen von Elementen:
Um ein Element hinzuzufügen. wählen Sie das gewünschte Feld in dem Pulldown-Menü 'Eine andere Spalte wählen ...'. Geben Sie die Spaltenüberschrift ein, die dann als Titel neben den Wert des Feldes angezeigt wird. Sie können auch einen Wert festlegen, der angezeigt wird, wenn für diesen Nutzer keine Daten gefunden werden. Klicken Sie auf 'Änderungen speichern'. Dann können Sie mit der Erstellung von neuen Elementen fortfahren.

Aktualisieren von Elementen:
Um die Überschrift oder fehlenden Text für ein Element zu ändern, aktualisieren Sie die Textfelder und klicken Sie dann auf 'Änderungen speichern'. Sie gelangen zurück zum Formular mit den entsprechenden neuen Werten.

Löschen von Elementen:
Klicken Sie auf das rote Kreuz neben dem Namen eines Elements, um dieses aus dem Überschriftenblock zu entfernen. Sie müssen Ihre Auswahl bestätigen.

Neuanordnen von Elementen:
Sie können Sie Reihenfolge der Elemente in der Liste ändern, indem Sie auf den Pfeil nach oben oder unten jeweils neben dem Element klicken.
