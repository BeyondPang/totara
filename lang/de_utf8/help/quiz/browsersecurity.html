Browser-Sicherheit

Diese Option bietet mehrere Möglichkeiten, 'Betrugsversuche' von Quizteilnehmern zu unterbinden. Dies ist jedoch nicht einfach. Was in einer Situation als 'Betrug' erachtet wird, kann in einer anderen Situation einfach nur eine effiziente Nutzung von Informationstechnologie sein. (Beispiel: die Fähigkeit, über eine Suchmaschine schnell Antworten zu finden.)

Beachten Sie auch, dass dies nicht nur ein technologisches Problem mit einer technischen Lösung ist. Betrugsversuche gab es schon lange vor den Computern. Computer machen zwar einige Aktionen, wie z.B. Kopieren und Einfügen, einfacher - sie machen es aber auch den Lehrern einfacher, Betrugsversuche zu erkennen, z.B. durch die Nutzung der Quizberichte. Die hier dargelegten Optionen sind nicht fehlersicher. Sie machen es den Teilnehmern zwar schwerer zu betrügen, sorgen aber gleichzeitig auch dafür, dass die Teilnahme an den Quizzen umständlicher wird.

Sie müssen auch andere Möglichkeiten in Betracht ziehen, wie Sie Betrugsversuche der Teilnehmer an Ihrem Quiz unterbinden können:

  • Sie können einen großen Fragenkatalog nutzen, bei dem das Quiz zufällig eine Auswahl von Fragen trifft, so dass die verschiedenen Teilnehmer unterschiedliche, aber ähnliche Fragen sehen.
  • Sie können die Option zur Vermischung der Antworten nutzen, so dass Option A nicht immer die korrekte Antwort auf Frage 1 ist.
  • Sie können Fragen stellen, bei denen die Teilnehmer die gegebenen Informationen analysieren müssen, und nicht nur Fakten wiedergeben müssen.

    Keine

    Für die Teilnehmer des Quizzes werden keine Hindernisse eingebaut.

    Vollbild-Popup mit gewisser JavaScript-Sicherheit

    Es besteht eine Beschränkung dahingehend, was das auf einem Web-Server laufende Quiz unternehmen kann, um die Handlungsmöglichkeiten der Quizteilnehmer an ihren jeweiligen Computern einzuschränken. Diese Option versucht jedoch, den maximalen Nutzen zu erzielen:

    • Das Quiz startet nur, wenn der Teilnehmer über einen JavaScript-fähigen Web-Browser verfügt.
    • Das Quiz wird in einem Vollbild-Popup angezeigt, das alle sonstigen Fenster überdeckt und über keine Navigationslemente verfügt.
    • Die Teilnehmer werden bestmöglich daran gehindert, Funktionen wie Kopieren und Einfügen zu nutzen.

    Erfordernis der Verwendung von Safe Exam Browser

    Diese Option wird nur angezeigt, wenn Ihr Administrator diese aktiviert hat.

    Safe Exam Browser ist ein individueller Web-Browser, der heruntergeladen und auf dem Computer, mit dem der Teilnehmer am Quiz teilnimmt, installiert werden muss. Die Einschränkungen für die Teilnehmer sind ähnlich zu den Einschränkungen des Popup-Fensters. Da Safe Exam Browser jedoch auf dem Computer des Teilnehmers läuft, kann diese Software noch wirksamere Schritte zur Beschränkung der Handlungsmöglichkeiten durchführen. Wenn Sie diese Option wählen:

    • Die Schüler können nur am Quiz teilnehmen, wenn sie Safe Exam Browser verwenden.
    • Das Browser-Fenster ist im Vollbildformat (ohne Navigationselemente).
    • Das Fenster kann erst geschlossen werden, wenn der Test übertragen wurde.
    • Tastenkürzel wie Win, Strg+Alt+Entf, Alt+F4, F1, Strg+P, Druck etc. sind deaktiviert.
    • Kopieren und Einfügen, sowie das Kontextmenü sind deaktiviert.
    • Das Wechseln zu anderen Anwendungen ist deaktiviert.
    • Das Surfen auf anderen Websites ist verboten.
