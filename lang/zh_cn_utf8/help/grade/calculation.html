计算

成绩薄中的计算遵循流行的电子表格程序中的公式/函数模式。它们以等号（=）开始，使用常见的数学运算符和函数，产生一个单一的数值输出。然后这个输出值将被用作您当前编辑的成绩项的计算值。

下面是计算支持的函数列表。逗号 (,) 用于分隔函数括号中的参数。

  • average([[item1]], [[item2]]...)：返回样本数据的平均值
  • max([[item1]], [[item2]]...)：返回一列参数中的最大值
  • min([[item1]], [[item2]]...)：返回一列参数中的最小值
  • mod(dividend, divisor)：计算除法运算的余数
  • pi()：返回 π 的值
  • power(base, power)：将一个数提升为另一个数的幂
  • round(number, count)：按预定义的精确度对一个数进行舍入
  • sum([[item1]], [[item2]]...)：返回所有参数的总和

同样也支持许多数学函数：

  • sin
  • sinh
  • arcsin
  • asin
  • arcsinh
  • asinh
  • cos
  • cosh
  • arccos
  • acos
  • arccosh
  • acosh
  • tan
  • tanh
  • arctan
  • atan
  • arctanh
  • atanh
  • sqrt
  • abs
  • ln
  • log
  • exp

您可以通过在公式中引用其他成绩项的 ID 来包含它们的值。ID 号由两对匹配的方括号括起，从而避免因 ID 号包含不常见的字符而混淆计算。例如，如果一个成绩项的 ID 为 Quiz.3，则在计算中，您将用 [[Quiz.3]] 表示此项。

计算区域下面是一个关于您课程中成绩类别和成绩项的树形目录。每一项或类别的总值旁边将显示可以在计算中使用的 ID（已经用方括号括起）。但是，由于 ID 是可选的，某些项可能尚无 ID。这些项的 ID 被一个表单字段代替，您可以在该字段中直接输入 ID，所以您可以在计算中使用它，而不必离开页面。尽管此表单不允许您编辑现有的 ID，但是，如果您要那样做，则需要离开此页面并直接编辑成绩项。

只要您指定了所需的 ID，则可以按“添加 ID”，然后页面将重新加载，并在相同的属性目录中显示您刚刚指定的 ID。然后便可以在计算中自由使用这些参数。

下面是一列计算实例，可以在您的成绩簿中使用：

  • =average([[Quiz.1]], [[Quiz.4]], [[Assignment.1]])
  • =average(max([[Quiz.1]], [[Quiz.4]], [[Assignment.1]]), min([[Quiz.1]], [[Quiz.4]], [[Assignment.1]]))
