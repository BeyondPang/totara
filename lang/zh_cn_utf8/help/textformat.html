格式化文本

在 Moodle 中编写文本时，有几种格式可供选择来生成文本，可选的格式取决于您的个人经验和您正在使用的浏览器类型。

通常您可以保持此设置为默认值，并且系统会按照您期望的方式工作。

1. Moodle 自动格式

此格式最适用于使用正常的 Web 页面进入（而非富文本 HTML 编辑器 ）。只需像发送电子邮件一样正常键入文本即可。

保存文本时，Moodle 会做很多事情以对您的文本进行自动格式化。

例如，以下形式的 URL

http://yahoo.com 或 www.yahoo.com 会被转换为链接。

将保留您的换行符，并且空白行处将开始新的段落。

笑脸代码如 :-) 会被自动转换为相应的图形。

您甚至可以根据需要嵌入 HTML 代码，并且代码将被保留。

2. HTML 格式

此格式假设文本是纯 HTML。如果您使用 HTML 编辑器来编辑文本，则这会是默认格式——工具栏中的所有命令都是用来产生 HTML 的。

即便您不是使用富文本 HTML 编辑器，您也可以在文本中使用 HTML 代码，它们的输出形式会和您希望的一样。

与 Moodle 自动格式不同，这种格式不会进行自动格式化。

3. 纯文本格式

当您希望包含大量代码或 HTML 时，并且希望显示的代码或 HTML 与您编写的一样，则此格式将非常有用。

此格式仍会转换空格和新行，但不会处理文本。

4. Markdown 文本格式

Markdown 格式试图仅靠使用和写电子邮件差不多的文本书写方式，使得键入格式良好的 XHTML 页面变得更容易。

要编写包含一些标题和列表但没有很多链接或图片的页面，使用该文本是非常合适的。
