报表格式

您可以为 Hot Potatoes 报表指定不同的输出格式。

报表格式

HTML
将以适合在 Web 浏览器中显示的 HTML 格式生成报表

Excel
报表将以“.xls”为后缀名的文件产生，此类文件可通过电子表格程序打开，例如 Microsoft Excel

文本
将报表生成为可通过文本编辑器打开的文本文件

编码

可能会提供编码列表，借助该列表，您能够根据电子表格程序的需要，将报表值强制转换为特定编码。 例如，Microsoft Excel 需要日语的“SJIS”编码。 站点管理员可以在 HotPot 模块的“设置”页中对可能的编码列表进行调整

数据打包

是
打包长型数据值以适应表格单元。这样可能会使某些行非常高

否
将不会打包数据值

显示摘要

是
在主报表中，问题和答案将用字母表示。系统将生成摘要，显示哪些字母代表哪些问题或答案

否
问题和答案将全部呈现在报表中，而不会再生成摘要
