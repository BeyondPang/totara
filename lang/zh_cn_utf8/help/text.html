撰写普通文本帮助

在 Moodle 中撰写普通文本的方式和您期望的方式差不多，但您仍可以在文本中加入“笑脸”、“URL 地址”和某些 HTML 标记。

笑脸（表情符号）

要在您的文本中加入以下小图标，只需键入相关代码即可。如果在看这些代码时您将头向左转，就会发现这些代码本身也都很象一张张小图片。

微笑 :-)
大笑 :-D
眨眼 ;-)
哭笑不得 :-/
深思 V-.
吐舌头 :-P
酷 B-)
满意 ^-)
瞪大眼睛 8-)

伤心 :-(

URL

任何以www.或http://开头的“词”都会被自动转换为可点击的链接。

例如：www.yahoo.com 或 http://curtin.edu

HTML 标记

您可以使用一组有限的 HTML 标记子集，以便在文本中添加重点。

HTML 标记产生
<b>粗体</b> 粗体文本
<i>斜体</i> 斜体文本
<u>下划线</u> 下划线文本
<font color=“green”>示例</font> 示例
<ul> <li>一个</li> <li>两个</li> </ul>
