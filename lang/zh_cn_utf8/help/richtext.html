富文本 HTML 编辑器提供了一个文字处理器界面，它嵌入在您的 Web 页面中，允许您以直观的方式编辑文本并生成普通的 HTML 代码。

除了对文本进行格式化之外，此编辑器还提供了一系列您可能会用到的其他功能。

从其他应用程序粘贴文本

您可以将富文本直接从其他 Windows 应用程序（如 Microsoft Word）中剪切和粘贴到此编辑器中，并且保留格式。只需使用 Web 浏览器中的剪切和粘贴菜单即可（或使用 Ctrl-C 和 Ctrl-V）。

插入图片

如果您在网站上发布了图片，并可以通过 URL 访问，则可以使用“插入图片”按钮将这些图片添加到您的文本中。

插入表格

要向文本添加布局，您可以使用工具栏中的“插入表格”按钮。

插入链接 /

要创建新链接，请首先键入要链接的文本。然后将其选中，并单击工具条中的链接按钮。键入要链接的 URL 即可完成！

插入笑脸（表情符号）

要在文本中插入这些小图标，请单击工具栏中的笑脸图标。系统会弹出一个对话框，在该对话框中，您可以选择以下笑脸图标。(另外，您也可以直接将相应的代码键入文本中，稍后显示文本时，会将它们转换成笑脸图标）。
