浏览器安全

此选项提供了各种方式来限制学生在测验时试图“作弊”的方法。然而，这并不是一个简单的问题。在某种情况下，这种行为被视为“作弊”，但在另一种情况下，这可能只是对信息技术的有效利用。（例如，利用搜索引擎快速查找答案的能力。）

同时请注意，这不仅仅只是与技术解决方案相关的技术问题。早在计算机出现之前，作弊行为就一直存在。当计算机使某些操作（如复制、粘贴）变得更为简便的同时，也让教师可以更加轻松地发现作弊行为 — 例如，使用测验报告。此处提供的选项并不是万无一失的。这些选项确实能够让学生难以作弊，但也会使学生们更加不愿尝试测验，因此并非万无一失。

您也应当考虑通过其他方法，使学生难于在测验时进行作弊：

  • 您可以使用大型题库，从中随机挑选问题作为测验题，这样，不同的学生看到的内容是不同的，但都是类似的问题。
  • 您可以使用答案重组选项，使问题 1 的正确答案并不总是 A 选项。
  • 您可以提问问题，要求学生分析给出的信息，而不仅仅只是复述事实。

    • 记住上述警告，以下是可用选项的描述。

    无

    没有在学生参与测验时设置障碍。

    具有某些 JavaScript 安全功能的全屏弹出窗口

    由于对 Web 服务器上运行的测验有一定限制，因此当学生坐在计算机前参与测验时其行为也会受到限制。但是，此选项还是可以起到一定限制作用的：

    • 只有当学生具备启用了 JavaScript 的 Web 浏览器时，测验才会开始。
    • 测验显示在全屏弹出窗口中，该窗口覆盖了所有其他窗口且没有导航控件。
    • 尽可能地阻止学生使用复制和粘贴之类的功能。

    需要使用安全测验浏览器

    只有当您的管理员启用了此选项时，它才会出现。

    安全测验浏览器是自定义的 Web 浏览器，必须下载并安装在学生用来参与测验的计算机上。针对学生的限制与弹出窗口中的限制类似，但是由于“安全测验浏览器”是一款运行在学生计算机上的软件，因此它可以更加有效地限制学生的操作。如果您选择此选项：

    • 如果学生使用的是“安全测验浏览器”，则学生将只能参与测验。
    • 浏览器窗口将为全屏显示（没有任何导航元素）。
    • 在提交测验之前无法关闭窗口。
    • 禁用 Win、Ctrl+Alt+Del、Alt+F4、F1、Ctrl+P 和 Printscreen 等快捷键。
    • 禁用复制、粘贴和上下文菜单。
    • 禁止切换到其他应用程序。
    • 禁止浏览其他网站。
