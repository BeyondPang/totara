使用 HTML 编辑器

编辑页面时，此选项支持您使用 HTML 编辑器（也称为编辑器）而不是简单的文本框。编辑器允许对文本进行简单的格式化以及添加一些“高级”功能（如链接）。

通过单击使用编辑器旁的框，可打开编辑器。编辑器是单个地与各项关联。例如，如果为响应 1打开编辑器，则只是为此项打开编辑器，而不是为所有响应框打开编辑器。

如果需要使用编辑器对某页的一些项进行编辑，请先单击使用编辑器框（这些框应当包含记号。）然后需要重新显示此页以激活编辑器的使用。通过单击此页页脚的重新显示页面按钮来完成此操作。

请注意，在使用编辑器时务必要小心。大多数情况下应当不会导致任何问题。但是，对于简答题和数字题的答案，不 建议使用编辑器。编辑器可以添加额外“不可见”的 HTML 标记，这些标记可以按预期安排停止对比。此外，分支表中的描述是在按钮中使用的，如果它们包含任何 HTML 标记，则会产生问题。对于仅用于显示的答案（如多选题的答案），则不会有任何问题。再次对响应项使用编辑器应当不会产生任何问题。

如果怀疑任何文本项有问题，安全的方法是关闭编辑器（通过移除使用编辑器框中的记号）。当重新显示此页时，可在文本框中看到“原始”文本。此时可根据需要编辑文本。实际上，为特定项重复打开和关闭编辑器应当不 会更改文本。

使用编辑器选项的“状态”与每一项一起保存。因此，如果重新编辑页面，曾使用编辑器的各项将再次出现在编辑器中。

最后，就浏览器内部的资源而言，编辑器是十分“昂贵”的。如果项目不需要使用编辑器，则使用标准文本框会更快速，而且很少导致资源问题。
