Par défaut les formulaires de Feedback créés sur votre page principale Moodle sont aussi disponibles sur l'ensemble du site and figureront dans tous les cours utilisant le bloc Feedback. Vous pouvez aussi forcer Feedback à figurer en en faisant un bloc permanent.

Vous pouvez cependant restreindre les cours pour lesquels un bloc Feedback apparraît en le mappant sur certains cours.
