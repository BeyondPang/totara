Imprimer enseignant

Pour imprimer le nom de l'enseignant sur le certificat, configurez le rôle de l'enseignant au niveau module. Faites-le si, par exemple, vous avez plus d'un enseignant par cours ou si vous avez plusieurs certificats pour ce cours et que vous souhaitez imprimer des noms d'enseignant différents pour chaque certificat. Cliquez sur modifier le certificat, puis cliquez sur l'onglet Rôles attribués locale. Puis attribuez le rôle d'enseignant (enseignant éditeur) au certificant (il n'ont PAS besoin d'être enseignants dans le cours--vous pouvez attribuer ce rôle à n'importe qui. Ces noms seront imprimés sur le certificat pour enseignant.
