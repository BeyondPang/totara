Type de certificat

C'est ici que vous déterminez la présentation du certificat. Le dossier de type de certificat contient quatre certificats par défaut :

De nouveaux types peuvent être ajoutés au dossier certificate/type. Les noms de dossiers et les nouvelles chaînes de caractères de langues pour le nouveau type doivent être ajoutés au fichier certificate/lang/en_utf8/certificate.php.

Paysage : imprime sur du papier A4 en orientation paysage.
Lettre paysage : imprime sur du papier format lettre en orientation paysage.
Portrait: imprime sur du papier A4 en orientation portrait.
Lettre portrait : imprime sur du papier format lettre en orientation portrait.

La plupart des langues s'imprimeront correctement, même avec des accents et caractères spéciaux. Cependant, si votre langue utilise ceux-ci, vous devrez choisir un type unicode. Ces types utilises des bibliothèques pdf différentes embarquant la police entière dans le pdf. Cela rendra les fichiers pdf plutôt imposants; il n'est donc pas conseillé d'utiliser ce type, sauf nécessité absolue.
