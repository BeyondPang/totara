Imprimer la note

Vous pouvez choisir tout objet de note de cours disponible dans le livre de notes pour imprimer la note de l'utilisateur reçue pour cet objet sur le certificat. Les objets de notes sont listés dans l'ordre dans lequels ils apparaissent dans le livre de notes.

Trois formats d'impression des notes sont disponibles si vous choisissez d'imprimer la note.

Note : Une fois que l'utilisateur a reçu le certificat, leur note y figurant ne peut PAS être modifiée.
