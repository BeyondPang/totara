Actividades dependientes

Elija aquí las actividades del curso y las calificaciones porcentuales mínimas que se deben alcanzar en cada una de ellas para obtener el certificado. Las actividades que se califiquen con puntos se convertirán en porcentaje para calcular la calificación necesaria.

Asimismo, puede elegir guardar el certificado para realizar una encuesta, un cuestionario o alguna actividad para proporcionar comentarios. Para estas actividades, configure el campo de calificación como "No".
