Foire aux questions sur les livres

Pourquoi deux niveaux seulement?
Deux niveaux suffisent généralement pour tous les livres, trois niveaux correspondraient à des documents mal structurés. Le module de livres est conçu pour la création de cours documents d'apprentissage de plusieurs pages. Il est en général conseillé d'utiliser le format PDF pour de longs documents. La manière la plus simple de créer des PDF est via les imprimantes virtuelles (voir PDFCreator, PDFFactory, Adobe Acrobat, etc..).

Les apprentis peuvent-ils modifier des livres ?
Seuls les enseignants peuvent créer et modifier des livres. Nous ne prévoyons pas d'implémenter de fonction d'édition pour les apprentis, mais vous êtes libre de créer quelque chose de similaire pour eux (Portfolio ?). La raison principale est que nous souhaitons garder le module livres le plus simple possible.

Comment rechercher dans les livres ?
Il n'existe actuellement qu'une possibilité, la fonction de recherche du navigateur sur une page imprimée. La recherche globale est maintenant possible uniquement sur les forums Moodle. Il serait agréable d'avoir la recherche globale implémentée pour toutes les ressources, dont les livres, des volontaires ?

Mes titres sont plus longs qu'une ligne.
Repenser votre titre ou demander à votre administrateur de modifier la largeur de la table des matières. Elle est définie globalement pour tous les livres sur la page de configuration du module.
