Aviso para el gestor

Si se selecciona la casilla de "Enviar aviso al gestor". el gestor del receptor del mensaje también recibirá una notificación que puede ser especificada en este campo.

El aviso para el gestor puede contener variables que pueden ser reemplazadas cuando el mensaje es enviado. Para más detalles acerca de qué variables serán sustituidas, vea sustitución de variables .
