Mitteilung für Manager

Wenn das Feld 'Mitteilung an Manager senden' markiert wurde, wird dem Manager des Meldungsempfängers ebenfalls eine Mitteilung übermittelt, die in diesem Feld festgelegt werden kann.

Die Mitteilung für Manager kann Variablen umfassen, die ersetzt werden, wenn die Meldung versendet wird. Für Details zu den Ersetzungen des Variablen, siehe Variablenersetzung .
