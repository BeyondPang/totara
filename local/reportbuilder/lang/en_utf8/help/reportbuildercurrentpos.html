Report Builder: Show by current position

This criteria is enabled by selecting the 'Show records from staff in the user's position' checkbox. If selected, then the report will show different records depending on which position the person viewing the report is in. This is determined by the settings in the 'Positions' tab in their user profile.

'Include records from child positions' determines which records the user sees:

  • Set to 'no':
    The viewer of the report sees all records for users who are currently in the same position as themselves.
  • Set to 'yes':
    The same as above, but the viewer of the report also sees records for users whos position lies below the viewer's position in the position hierarchy.
